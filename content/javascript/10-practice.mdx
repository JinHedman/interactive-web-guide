<Exercise
  title="FizzBuzz from 1 to 20"
  goal="Loop from 1 to 20 and log Fizz, Buzz, FizzBuzz, or the number, following the classic rules."
  steps={[
    "Create fizzbuzz.js and write a for loop that counts i from 1 to 20.",
    "Inside the loop, check the both-multiples case first: if i is divisible by 3 AND by 5, log 'FizzBuzz'.",
    "Else if i is divisible by 3, log 'Fizz'. Else if i is divisible by 5, log 'Buzz'.",
    "Otherwise log the number itself.",
    "Use the modulo operator i % 3 === 0 to test divisibility. Run it and check the pattern.",
    "Commit in GitHub Desktop with the message 'Add FizzBuzz drill'."
  ]}
  expected="Twenty lines: 1, 2, Fizz, 4, Buzz, Fizz, 7, 8, Fizz, Buzz, 11, Fizz, 13, 14, FizzBuzz, ... up to 20."
  solutionLang="js"
  solution={`for (let i = 1; i <= 20; i++) {
  // Check the combined case first — order matters.
  if (i % 3 === 0 && i % 5 === 0) {
    console.log("FizzBuzz");
  } else if (i % 3 === 0) {
    console.log("Fizz");
  } else if (i % 5 === 0) {
    console.log("Buzz");
  } else {
    console.log(i);
  }
}`}
/>

<Exercise
  title="Fix this: the loop that never stops (and reads too far)"
  goal="Two loops are broken — one runs forever, one reads past the end of the array. Find both bugs, fix them, and explain each in a comment."
  steps={[
    "Create loop-fix.js and paste the buggy code from the solution's first version (or read it below before revealing).",
    "Bug 1: the while loop logs the same value forever because the counter is never updated. Add the missing step so it moves toward the exit.",
    "Bug 2: the for loop uses i <= nums.length, so on the last pass it reads nums at an index that does not exist (undefined). Change the condition so it stops at the last real index.",
    "Run it: the while loop should count down and stop, and the for loop should print only real values — no undefined.",
    "Add a one-line comment on each fix saying what went wrong."
  ]}
  expected="The while loop prints 3, 2, 1 and stops; the for loop prints 10, 20, 30 with no trailing undefined."
  solutionLang="js"
  solution={`// Bug 1 fixed: without count-- the condition stays true forever (infinite loop).
let count = 3;
while (count > 0) {
  console.log(count);
  count--; // moves toward the exit — without this the page would freeze
}
// 3, 2, 1

const nums = [10, 20, 30];
// Bug 2 fixed: i < nums.length, not <=. With <= the last pass read nums[3],
// which does not exist, logging undefined.
for (let i = 0; i < nums.length; i++) {
  console.log(nums[i]);
}
// 10, 20, 30`}
/>

<Exercise
  title="Find the largest number without Math.max"
  goal="Walk an array with a loop and a running 'best so far' variable to find and log the maximum."
  steps={[
    "Create largest.js and declare const nums = [12, 7, 25, 3, 19];.",
    "Declare let largest = nums[0]; — start with the first value as the best guess.",
    "Use a for...of loop over nums. If the current value is greater than largest, update largest.",
    "After the loop, log largest. It should be 25.",
    "Stretch: in the same loop, track a let smallest the same way and log it too (it should be 3)."
  ]}
  expected="The console logs 25 (the largest). With the stretch, it also logs 3 (the smallest)."
  solutionLang="js"
  solution={`const nums = [12, 7, 25, 3, 19];

let largest = nums[0];
let smallest = nums[0];

for (const n of nums) {
  if (n > largest) {
    largest = n;
  }
  if (n < smallest) {
    smallest = n;
  }
}

console.log(largest);  // 25
console.log(smallest); // 3`}
/>

<Exercise
  title="Average of an array (a reusable function)"
  goal="Write a function that takes an array, sums it with a loop, and returns the mean — then log one call."
  steps={[
    "Create average.js and declare function average(numbers) { ... }.",
    "Inside, start let sum = 0; and loop over numbers with for...of, adding each to sum.",
    "Return sum / numbers.length — return the value, do not console.log inside the function.",
    "Call it: console.log(average([4, 8, 15, 16, 23, 42]));.",
    "Run it. Because the function returns, the caller is what logs the result.",
    "Reveal the solution only after your version prints 18 (the mean of those six numbers)."
  ]}
  expected="The console logs 18 — the average of 4, 8, 15, 16, 23, 42."
  solutionLang="js"
  solution={`function average(numbers) {
  let sum = 0;
  for (const n of numbers) {
    sum += n;
  }
  return sum / numbers.length; // return, so the caller can use the result
}

console.log(average([4, 8, 15, 16, 23, 42])); // 18`}
/>
